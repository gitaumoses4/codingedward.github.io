Work Data structures and algorithms
Duration 2016 - Now

These algorithms involve:

  • Divide and conquer - Quicksort, Mergesort.
  • Data structures - BSTs, Hash tables, Disjoint sets, Red Black and AVL trees.
  • Sorting and order statistics - Heapsort, Quicksort, Bucket sort, Radix Sort etc.
  • Dynamic programming - Rod cutting, Longest common subsequence.
  • Greedy Algorithms - Huffman codes, Activity selection problem.
  • Graph Algorithms - BFS, DFS, Topological sort, Kruskal's and Prim's MST.

  • Maximum flow - Ford-Fulkerson, Maximum bipartite matching, Push-relabel.
